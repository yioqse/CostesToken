# 🧮 IA Cost Calculator (Local Edition)

Sistema modular para la estimación de costes de APIs de Inteligencia Artificial utilizando procesamiento de lenguaje natural (NLP) local para el conteo de tokens.

## 🚀 Características
- **Privacidad:** El conteo de tokens ocurre en local mediante `Transformers`.
- **Arquitectura:** Desacoplada y lista para integración con APIs o Microservicios.
- **Calidad:** Suite de tests unitarios e integración con GitHub Actions (CI).

## 📂 Estructura
- `/calculadora`: Lógica de negocio y motor de tokens.
- `/services`: Pipeline de procesamiento.
- `/docs`: Documentación técnica detallada.
- `/tests`: Pruebas de regresión y calidad.
